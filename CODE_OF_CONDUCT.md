# Code of Conduct

PlayMore is a small open-source project. Be respectful. Don't be a jerk.

## What's expected

- Be kind and constructive in code reviews and issue discussions.
- Assume good intent. Disagree about ideas, not people.
- If you disagree with a decision, explain your reasoning. The maintainer makes final calls.

## What's not OK

- Personal attacks, harassment, or discriminatory language.
- Deliberately disruptive behavior in issues or PRs.
- Spam, off-topic promotion, or platform abuse.

## Reporting

If someone is making the project unpleasant, contact the maintainer privately via GitHub. Repeated violators will be blocked from the repo.

This is intentionally short. If you've been on the internet, you know the rules.
